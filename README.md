# Spice Haven Food 🍔

A food ordering UI built with HTML, Tailwind CSS, and JavaScript.<br>

## Features<br>
Animated Landing Page<br>
Smooth Burger Transition<br>
Food Menu Cards<br>
Search Bar UI<br>
Category Navigation<br>
Tailwind CSS Styling<br>
Interactive Hover Effects<br>

## 📱 Pages<br>
1.Landing Page<br>
● Gradient background<br>
● Animated burger image<br>
● Smooth transition to menu page<br>

2.Menu Page<br>
● Food cards<br>
● Search section<br>
● Categories<br>
● Ratings<br>
● Footer navigation<br>

## 🛠️Technologies Used<br>
● HTML5
● Tailwind CSS
● JavaScript
